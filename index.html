HTML & CSS Basics Summary

HTML Summary

HTML (Hyper-Text Markup Language) is used to define the content, meaning and structure of our webpage. You don't use it for styling purposes. Use CSS for that instead.

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening, content and a closing tag
  • You can also have void (empty) elements like images
  • You can also configure elements using attributes
  • There is a long list of elements, use semantic elements instead of divs

Learn all about available HTMP elements on the MDN elements reference page.

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content.

  • Styles are assigned via property-value pairs
  • You can assign styles via the "styles" attribute
  • To avoid duplication, use global styles by linking external style sheet using "link" element in head section of HTMP page
  • When working with CSS, one should be familiar with Box Model, Inheritance and Specificity
  • CSS is also used for "Page Layouts" using Flexbox and Grid

Learn more about all available CSS properties and values on the MDN css reference page.
